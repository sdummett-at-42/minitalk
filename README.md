# Welcome to minitalk

> The purpose of this project is to code a small data exchange program using UNIX signals.

## Install

```sh
git clone git@github.com:sdummett/minitalk.git ...
```

## Usage

```sh
make
./server
./client $(pidof server) [string to send]
```
## Bonus
```sh
make bonus
./server
./client $(pidof server) [string to send]
```
